First-time setup — connect Google Sheets backend

🌐 Google Sheets Connection

This system uses Google Sheets as the live database. Paste your Apps Script Web App URL to connect.

Open Google Sheets → Extensions → Apps Script
Paste the Apps Script code (see Settings after setup)
Deploy → New Deployment → Web App → Execute as: Me → Access: Anyone
Copy the deployment URL and paste below
Must end with /exec
Found in the sheet URL: /spreadsheets/d/SHEET_ID/edit

🔐 Create Admin Account

This is the master admin account. Keep the password safe.

✅ Confirm Setup

Review your configuration before finishing.

⚠️
Make sure your Apps Script is deployed and accessible before finishing. The admin account will be saved to your Google Sheet.
